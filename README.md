# AI Operational Memory

A continuously running operational intelligence and memory layer for AI agent ecosystems.

## Philosophy

> Observe automatically. Decide manually.

This system helps operators:
- understand evolving AI infrastructures
- reconstruct workflows automatically
- track LLM usage and costs
- preserve operational memory
- generate actionable recommendations

The system is strictly advisory-only:
- no autonomous modifications
- no infrastructure control
- no self-modifying behavior

## Current Status

Early architecture and bootstrap phase.

## Technical Direction

- Python
- FastAPI
- Docker Compose
- SQLite initially
- Structured logging
- Read-only intelligence model
